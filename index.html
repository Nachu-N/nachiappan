Machine Learning Based Anomaly Detection in Cybersecurity

Wiley IET Information Security | Under Review

The study investigates anomaly detection in cybersecurity, highlighting the crucial role that anomaly detection plays in defending critical infrastructure against sophisticated cyberattacks. An essential strategy is anomaly detection because traditional signature-based detection techniques frequently fail to discover novel attack routes. Cutting edge technologies like Artificial Intelligence (AI) methods, such as Machine Learning (ML) and DeepLearning (DL), makes it possible to analyze enormous volumes of data in real time, spot minute irregularities, and accurately forecast possible security threats. The usefulness of algorithms like Support Vector Machines (SVM), k-Nearest Neighbors (kNN), Random Forests (RF), and neural networks is highlighted in this study as it examines the application of these approaches in malware protection, network intrusion detection, and phishing detection. It discusses the use of supervised, unsupervised, and reinforcement learning methodologies and reports performance measures such as F1-score, accuracy, precision, and recall. This investigation also explores important research issues, such as data quality, computing efficiency, and the integration of anomaly detection systems with current security frameworks. It emphasizes how important it is to create effective data structures and algorithms in order to manage continuous data streams with little computational cost. The study finishes with strategic recommendations for building resilient and flexible cybersecurity defenses using AI-driven anomaly detection, thereby boosting critical infrastructure security in a constantly changing threat environment. The challenges faced in anomaly detection and factors influencing it in cybersecurity are discussed towards the end with directionsfor future research

ITRAC: Intelligent Tracking and Real-Time Analysis on Cloud

TENCON 2025 | Kota Kinabalu, Malaysia

This research explores real-time monitoring of diverse assets using AIoT, leveraging edge-cloud collaboration for enhanced responsiveness and intelligence. RFID sensors are employed to validate edge computing functionality in tracking and anomaly detection. RFID tag reads are processed on the edge, which collect temporal data such as tag frequency and signal strength with precise timestamps to identify irregularities. A compound AI system integrates an LSTM model for data encoding and normalization, and an Isolation Forest for outlier detection on critical features. These models are containerized using Docker to enable consistent deployment across edge devices and cloud platforms. Processed RFID data is transmitted from a Raspberry Pi to a private OpenStack cloud via MQTT using QoS level 1, ensuring reliable, low-latency anomaly reporting. A lightweight dashboard interface enables real-time visualization and monitoring of anomalies. The proposed system architecture was evaluated on real time asset tracking using RFID and the machine learning workflow was trained using three setups 1. LSTM has shown an F1 score: 0.84, 2. Isolation forest has shown F1 score: 0.87, and 3. LSTM + Isolation Forest has shown F1 score: 0.84. The hybrid model balanced both, making it suitable for real-time RFID anomaly detection. The system is scalable, secure, and adaptable for applications in logistics, healthcare, and industrial asset tracking.
